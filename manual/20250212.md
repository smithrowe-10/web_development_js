## 로또 번호에 대해 생각해봐야 할 점
1. 숫자가 6개, 즉 변수 6개를 준비 -> 배열로 처리 될 것
2. 1 ~ 45 까지의 임의의 자연수를 생성
3. 이전에 뽑은숫자와 동일한 값이라면 다시 임의의 자연수를 생성해야 함.
4. 다르다면 변수 1 ~ 변수 6까지 대입 -> 배열의 0번지부터 5번까지의 대입으로 처리가능할 것

Math.random()을 사용했습니다 => return값이 0초과 1 미만의 난수
